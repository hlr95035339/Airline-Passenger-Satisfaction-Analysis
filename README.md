# Airline-Passenger-Satisfaction-Analysis

## Overview
This project analyzes airline passenger satisfaction, focusing on arrival delays, cabin class, and customer loyalty.

## Business Insights
- Delays strongly reduce satisfaction, especially in economy class.
- Loyal customers remain more resilient, highlighting the importance of retention programs.
- Operational KPI target: Keep average delays below 15 minutes.
